Options for protoc.

com.google.closure.plugin.proto.ProtoOptions

<proto>

<descriptorSetFile>FilePath of output descriptor set file.
<exclude>StringAdd a path patterns to exclude specified as ANT-directory-scanner-style patterns like **/*.ext.
<id>StringIf not supplied, autogenerated.

Compilers that may be run multiple times with different options may use the ID to disambiguate outputs.

<include>StringAdd a path patterns to include specified as ANT-directory-scanner-style patterns like **/*.ext.
<javaOnly>StringProtobuf packages to only compile only to Java. If a package appears on the {@link #jsOnly} and {@link #javaOnly} lists then it will be compiled to both.
<jsOnly>StringProtobuf packages to only compile only to JS. If a package appears on the {@link #jsOnly} and {@link #javaOnly} lists then it will be compiled to both.
<protobufVersion>StringProtobuf version to compile schema files for. If omitted, version is inferred from the project's depended-on com.google.com:protobuf-java artifact, if any. (If both are present, the version must match.)
<protocExec>FilePath to existing protoc to use. Overrides auto-detection and use of bundled protoc.
<source>SourceRootBuilderSource file roots.
<testDescriptorSetFile>FilePath of output descriptor set file or test protos.
<testSource>SourceRootBuilderTest file roots.
</>
